# Product

## Overview

A personal productivity dashboard — a browser-based start page that combines a greeting, focus timer, to-do list, and quick links in one clean interface.

## Target Users

Individual users who want a lightweight, customizable new-tab dashboard with no sign-up or backend required.

## Core Features

- **Greeting** — shows current time, date, and a time-based greeting with a custom user name
- **Focus Timer** — Pomodoro-style timer with configurable duration, start/stop/reset controls
- **To-Do List** — add, edit, mark done, delete, sort tasks; duplicate prevention; persisted in localStorage
- **Quick Links** — user-defined shortcut buttons to favorite websites, persisted in localStorage
- **Light / Dark Mode** — toggle with preference saved to localStorage

## Constraints

- No frameworks, no backend, no build tools
- All data stored client-side via localStorage
- Single HTML file, one CSS file, one JS file
- Must work in Chrome, Firefox, Edge, and Safari
- Deployable as a static site via GitHub Pages
