# CodingCamp-4June26-yoeljoy

A personal productivity dashboard built with pure HTML, CSS, and Vanilla JavaScript — no frameworks, no build tools.

## Features

- **Greeting** — live clock, date, and time-based greeting with your custom name
- **Focus Timer** — configurable Pomodoro timer (1–60 min) with start / stop / reset
- **To-Do List** — add, edit, complete, delete, and sort tasks; duplicate prevention; persisted in localStorage
- **Quick Links** — shortcut buttons to your favourite sites, persisted in localStorage
- **Light / Dark Mode** — toggle with preference saved across sessions

## File Structure

```
├── index.html
├── css/

```

## Usage

Open `index.html` directly in any modern browser — no server or installation needed.

## Deployment

Push to GitHub and enable **GitHub Pages** from the repository settings (source: `main` branch, root `/`).
